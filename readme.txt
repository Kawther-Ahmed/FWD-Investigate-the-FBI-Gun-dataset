Investigating and analyzing the FBI Gun dataset using Python and data visualization techniques. The repository contains two main files: "Investigate_a_Dataset.html" and "Investigate_a_Dataset.ipynb". These files contain the code and documentation for the investigation and analysis performed on the dataset. The project utilizes various Python libraries such as Pandas, and Matplotlibto extract insights and create visualizations from the dataset. The investigation aims to explore patterns, trends, and relationships within the FBI Gun dataset, providing valuable insights into firearm background checks in the United States.
